# Java Language Basics & Syntax

A quick reference guide for Java syntax, initialization, and best practices.

## 1. Syntax Basics
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Ready for the 90-day journey!");
    }
}
```

## 2. Variable Initialization
```java
int number = 10;
String text = "DSA";
boolean isRunning = true;
int[] arr = new int[10];
```

## 3. Core Concepts
- Object-Oriented Programming (Classes, Objects, Inheritance)
- Collections Framework (ArrayList, HashMap, HashSet)
- Exception Handling

> [!IMPORTANT]
> Always follow **PascalCase** for Class names and **camelCase** for variable/method names.
